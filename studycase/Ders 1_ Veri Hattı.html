Algoritma Mantığı

Yapay zeka modelleri eğitilmeden önce, ham verilerin bilgisayar tarafından doğru ayrıştırılması gerekir. Amacımız metinsel (kategorik) veriler ile matematiksel (sayısal) verileri birbirinden ayırmaktır.

Kodlama Yol Haritası

Yandaki editörün içine sırayla şu mantığı kurmanız bekleniyor:

  1. Eğer gelen verinin tipi metin ("string") ise, geriye "kategorik" değerini döndür.
  2. Değilse, eğer verinin tipi sayı ("number") ise, geriye "sayisal" değerini döndür.
  3. Yukarıdaki iki duruma da uymuyorsa, geriye "tanimsiz" değerini döndür.

Kullanılacak Sözdizimi

if (typeof deger === "string") {
    return "kategorik";
} else if (typeof deger === "number") {
    // kodunuz...
}
