Model Değerlendirme: Doğruluk Skoru (Accuracy) Metriği

Doğru Hesaplama: 0 / 6

Algoritma Mantığı

Bir yapay zeka modelini eğitmek kadar onun başarısını ölçmek de kritik bir adımdır. Sınıflandırma algoritmalarında en sık başvurulan metrik **Doğruluk Skorudur (Accuracy)**. Doğruluk skoru, modelin yaptığı doğru tahmin sayısının, test edilen toplam veri sayısına bölünmesiyle hesaplanır.

Kodlama Yol Haritası

Yandaki editörün içine sırayla şu mantığı kurmanız bekleniyor:

  1. Gelen `tahminler` ve `gercekler` parametrelerinin ikisi de aynı uzunlukta tam sayı dizileridir (Örn: `[1, 0, 1]`). Başlangıçta `dogruTahmin` adında `0` değerine sahip bir sayaç değişkeni oluşturun.
  2. Bir `for` döngüsü kurarak dizilerin üzerinde dönün. Eğer döngü indeksindeki `tahminler[i]` değeri, `gercekler[i]` değerine eşitse doğru tahmin sayacını 1 artırın.
  3. Döngü bittikten sonra, toplam doğru tahmin sayısını dizinin toplam eleman sayısına (`tahminler.length`) bölerek oran bilgisini (0 ile 1 arasında) bulun ve geriye `return` edin.

Kullanılacak Yapı

let dogru = 0;
for (let i = 0; i < tahminler.length; i++) {
    if (tahminler[i] === gercekler[i]) {
        dogru++;
    }
}
return dogru / tahminler.length;
