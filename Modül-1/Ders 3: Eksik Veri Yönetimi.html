Eksik Veri (Null) Tespiti ve İmputasyon Algoritması

Temizlenen Veri: 0 / 6

Algoritma Mantığı

Yazılım hataları veya kullanıcıların formları boş bırakması nedeniyle veri tabanına `null` (boş) veriler kaydolabilir. Yapay zeka modelleri eksik verilerle eğitilemez. Bu yüzden boş alanları mantıklı varsayılan değerlerle doldurma işlemine **imputasyon** denir.

Kodlama Yol Haritası

Yandaki editörün içine sırayla şu mantığı kurmanız bekleniyor:

  1. Gelen sepet nesnesinin tutar bilgisini kontrol edin. Eğer `sepet.tutar` değeri `null` veya `undefined` ise, bu değeri şirketin belirlediği taban değer olan `500` sayısı ile güncelleyin.
  2. Değer null değilse mevcut tutarı aynen koruyun.
  3. İşlem sonunda güncellenmiş veya değiştirilmemiş olan tüm `sepet` nesnesini geriye `return` edin.

Kullanılacak Sözdizimi

if (sepet.tutar === null) {
    sepet.tutar = 500;
}
return sepet;
